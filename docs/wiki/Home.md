# Immersive Health System — Wiki

> **🛑 Project closed.** This wiki is the final documentation snapshot.

Welcome to the IHS wiki. The plugin adds a modular disease, medicine and
laboratory system to PaperMC servers. The wiki is split between
**player**, **admin** and **developer** sections.

## Players

- [Player Guide](Player-Guide.md) — what you'll see in-game, how to get
  sick, how to recover, what the tranquilizer does.

## Server admins

- [Admin Guide](Admin-Guide.md) — installation, commands, permissions,
  configuration files.
- [Resource Pack](Resource-Pack.md) — how IHS delivers its visuals and
  sounds (CraftEngine / ItemsAdder / Nexo / Oraxen / manual).
- [Compatibility](Compatibility.md) — server / client / Java version
  matrix.

## Developers

- [Developer API](Developer-API.md) — using `IHSApi` from your own
  plugin, plus the available Bukkit events.
- [Forking](Forking.md) — renaming the plugin, swapping textures,
  publishing your own derivative.
